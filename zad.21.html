Co znaczy skrót ID? ID to unikalny identyfikator, który służy do oznaczenia pojedynczego elementu na stronie.
Jaki jest cel stosowania ID? ID umożliwia jednoznaczne wskazanie elementu do stylizacji CSS, manipulacji w JavaScript lub nawigacji w dokumencie.
Jakie jest znaczenie znaku # Znak # w CSS oznacza selektor ID i służy do wyboru elementu o danym identyfikatorze.
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? W CSS ID wywołuje się przez poprzedzenie nazwy identyfikatora znakiem #, np. #mojeID { ... } — to mówi, że styl dotyczy elementu z tym ID.
Uwagi na temat stosowania ID (trzy konieczne wyliczenia) 1. ID musi być unikalne na stronie, nie powtarza się.
2. Stosowanie ID jest ważne do precyzyjnej stylizacji i skryptów.
3. Nadmierne używanie ID może utrudnić utrzymanie kodu, dlatego warto używać ich z umiarem.
